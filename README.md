# Web-Application-Security-Scanner
A Python-based web application security scanner that identifies search fields and cookies, tests multiple SQL injection payloads, analyzes responses, performs risk scoring, and generates detailed text report for vulnerability assessment. Features : input field enumeration, cookie analysis, response-based detection, automated report generation.




How to Run the Web Application Security Scanner:-

Prerequisites:

Python installed on your system
Visual Studio Code (VS Code)

Steps:

1. Create the Project Folder
Create a folder named:
Web Application Security Scanner

2. Add the Scanner File
Copy the crawler.py file and paste it into the Web Application Security Scanner folder.

3. Create a Reports Folder
Inside the Web Application Security Scanner folder, create another folder named:
Web Scanner Reports

4. Open the Project in VS Code
Open the Web Application Security Scanner folder in Visual Studio Code.

5. Configure the Target URL
Open the crawler.py file and set the URL of the web application you want to scan in the start_url variable.
Example:
start_url = "https://example.com"

6. Run the Scanner
Execute the crawler.py script.

7. View the Generated Report
After the scan is completed, the report will be automatically generated and saved in the Web Scanner Reports folder.

Output:

All scan reports can be found in:
Web Application Security Scanner/Web Scanner Reports
